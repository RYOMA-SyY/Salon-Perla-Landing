# SpaLab Website - Complete Style Description & Design Prompt

## Overview
This is a sophisticated, elegant beauty and spa website design with a clean, modern aesthetic. The design emphasizes luxury, tranquility, and wellness through a carefully curated color palette, elegant typography, and high-quality imagery.

---

## Typography System

### Primary Fonts
- **Body Text Font**: `'Raleway', 'Arial', sans-serif`
  - Font size: 14px
  - Line height: 1.6
  - Color: #868686 (medium gray)

- **Headings Font**: `'Lato', 'Arial', sans-serif`
  - Font weight: 400 (regular)
  - Line height: 1.2
  - Color: #181818 (dark gray/black)
  - Heading sizes:
    - Small: 15px
    - Medium: 19px
    - Large: 29px
    - XL: 39px
    - XXL: 59px

- **Decorative Font**: `"Indie Flower", Sans-serif` (used for special decorative elements)

### Typography Characteristics
- Clean, sans-serif modern typefaces
- Generous line spacing for readability
- Clear hierarchy through size differentiation
- Headings use lighter font weight (400) for elegance

---

## Color Palette

### Primary Colors
- **Primary Text**: `#181818` (Dark Gray/Charcoal Black)
  - Used for headings and primary text elements
  - Creates strong contrast and readability

- **Body Text**: `#868686` (Medium Gray)
  - Used for paragraph text and secondary content
  - Provides softer, less harsh reading experience

- **Accent Color**: `#5c3474` (Purple/Violet)
  - Used for link hover states
  - Adds elegant, luxurious touch
  - Creates visual interest and interactivity

### Secondary Colors
- **Background**: `#fff` (White)
  - Primary background color
  - Clean, minimal base

- **Secondary Background**: `#263944` (Dark Blue-Gray)
  - Used for contrast sections
  - Creates depth and visual separation

- **Divider/Icon Color**: `#69727d` (Gray-Blue)
  - Used for dividers, icons, and UI elements
  - Subtle, professional appearance

- **Dark Accent**: `#0c0d0e` (Near Black)
  - Used for borders and deep contrasts

### Color Usage Strategy
- High contrast between text and backgrounds for accessibility
- Subtle color transitions for elegant feel
- Purple accent adds sophistication without being overwhelming
- White space and neutral grays create breathing room

## Layout & Structure

### Container System
- **Max Width**: 1600px (main content containers)
- **Container Type**: Flexbox-based responsive containers
- **Alignment**: Centered with `margin: 0 auto`
- **Section Spacing**: Relative positioning for flexibility

### Section Dimensions
- **Hero Sections**: Minimum height 600px
- **Responsive Breakpoints**:
  - Desktop Large: 1540px and above
  - Desktop: 1280px - 1540px
  - Tablet: 1024px - 1280px
  - Mobile Large: 767px - 1024px
  - Mobile: 480px - 767px
  - Mobile Small: Below 480px

### Grid Systems
- **Column Layouts**: 25%, 33.33%, 50%, 65%, 100% widths
- **Responsive Grid**: Adapts from 4 columns (desktop) to 1 column (mobile)
- **Image Boxes**: 4-column grid (25% each) on desktop, 2-column (50%) on tablet, 1-column (100%) on mobile

---

## Responsive Design Philosophy

### Mobile-First Approach
- Flexible grid systems
- Responsive typography
- Image optimization
- Touch-friendly spacing

### Breakpoint Strategy
1. **Large Desktop (1540px+)**: Full 4-column layouts, maximum width containers
2. **Desktop (1280-1540px)**: Slightly reduced columns, maintained proportions
3. **Tablet (1024-1280px)**: 2-column layouts, adjusted spacing
4. **Mobile Large (767-1024px)**: 2-column to 1-column transitions
5. **Mobile (480-767px)**: Single column layouts, optimized spacing
6. **Mobile Small (<480px)**: Full-width elements, minimal margins
