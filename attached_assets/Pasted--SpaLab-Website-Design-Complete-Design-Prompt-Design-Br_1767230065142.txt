# SpaLab Website Design - Complete Design Prompt

## Design Brief

Create an elegant, luxurious beauty and spa website with a modern, tranquil aesthetic. The design should convey sophistication, relaxation, and premium quality through carefully selected typography, a refined color palette, and high-quality imagery.

---

## Typography Specifications

### Primary Typeface (Body Text)
- **Font Family**: Raleway, Arial, sans-serif
- **Font Size**: 14px
- **Line Height**: 1.6
- **Color**: #868686 (medium gray)
- **Weight**: Regular (400)
- **Usage**: All body text, paragraphs, and general content

### Heading Typeface
- **Font Family**: Lato, Arial, sans-serif
- **Font Weight**: 400 (regular, not bold)
- **Line Height**: 1.2
- **Color**: #181818 (dark charcoal)
- **Size Scale**:
  - Small: 15px
  - Medium: 19px
  - Large: 29px
  - Extra Large: 39px
  - Extra Extra Large: 59px
- **Usage**: All headings (h1-h6), titles, section headers

### Decorative Typeface
- **Font Family**: "Indie Flower", Sans-serif
- **Usage**: Special decorative text elements, accents

## Color Palette

### Primary Colors
```
Primary Text:        #181818  (Dark Charcoal)

Background:          #ffffff  (Pure White)
```

### Secondary Colors
```
Secondary Background: #263944  (Dark Blue-Gray)
Divider/Icon Color:   #69727d  (Gray-Blue)
Dark Accent:          #0c0d0e  (Near Black)
```

### Color Application Rules
- Use #181818 for all headings and primary text elements
- Use #868686 for body text and secondary content

- Apply #69727d for dividers, icons, and subtle UI elements

---

## Layout & Structure

### Container System
- **Maximum Width**: 1600px for main content containers
- **Layout Method**: Flexbox-based responsive containers
- **Alignment**: Center-aligned with automatic margins
- **Section Structure**: Full-width sections with contained content

### Section Specifications
- **Hero Sections**: Minimum height 600px
- **Background Images**: Full-bleed, cover size, center positioned
- **Overlays**: Positioned at 80% horizontal, 0% vertical (or custom as needed)
- **Blend Mode**: Multiply for overlay effects
- **Transitions**: 0.3s duration for all interactive states

### Grid System
- **Desktop (1540px+)**: 4-column layouts (25% each column)
- **Desktop (1280-1540px)**: 4-column or 3-column layouts
- **Tablet (1024-1280px)**: 2-column layouts (50% each)
- **Mobile Large (767-1024px)**: 2-column to 1-column
- **Mobile (480-767px)**: Single column (100% width)
- **Mobile Small (<480px)**: Single column, full-width elements

### Column Width Examples
- 25% (quarter width)
- 33.33% (third width)
- 35% (narrow column)
- 48% (near half)
- 50% (half width)
- 52% (slightly more than half)
- 58% (wide column)
- 65% (wide column)
- 100% (full width)

---

## Visual Design Elements

### Background Images
- Use high-quality spa/beauty photography as full-bleed backgrounds
- Implement background overlays with positioned images (woman figures, decorative elements)
- Apply mix-blend-mode: multiply for sophisticated overlay effects
- Use background-size: cover for main backgrounds
- Use background-size: contain for overlay images
- Position backgrounds at center center, or custom positions (80% 0%, 87% 100%)

### Decorative Graphics
- Include elegant flower icons for accent elements
- Use feather graphics for decorative touches
- Maintain transparent backgrounds for PNG graphics
- Position decorative elements strategically without overwhelming content

### Image Treatment
- All images: max-width 100%, height auto (responsive)
- Vertical alignment: middle
- Display: inline-block
- Maintain aspect ratios
- Use high-resolution imagery optimized for web

---

## Interactive Elements

### Links & Navigation
- Default link color: #181818 (matches primary text)
- Hover state color: #5c3474 (purple accent)
- Text decoration: none (remove underlines)
- Smooth color transition on hover

### Buttons & CTAs
- Clean, minimal styling
- Focus on typography over heavy borders
- Subtle hover effects
- Maintain elegant aesthetic

### Drop Caps (When Used)
- Font size: 50px
- Stacked style: Background #69727d, text white
- Framed style: Color #69727d, 3px solid border, transparent background
- Float: left
- Margin: top 8px

---

## Spacing & Composition

### Global Reset
- Box-sizing: border-box (applied to all elements)
- Margin: 0
- Padding: 0

### Spacing Principles
- Generous white space throughout
- Strategic use of negative margins for overlapping effects (e.g., -40px)
- Padding and margins should create breathing room
- Maintain visual hierarchy through spacing

### Content Spacing
- Section margins: Adjust based on content needs
- Column spacing: Controlled via CSS custom properties
- Element spacing: Sufficient gap between interactive elements

---

## Responsive Design Specifications

### Breakpoint Strategy
1. **Large Desktop**: 1540px and above
   - Full 4-column layouts
   - Maximum 1600px containers
   - All decorative elements visible

2. **Desktop**: 1280px to 1540px
   - Maintained proportions
   - Slight column adjustments
   - Full functionality

3. **Tablet Landscape**: 1024px to 1280px
   - 2-column layouts
   - Adjusted spacing
   - Maintained readability

4. **Tablet Portrait / Mobile Large**: 767px to 1024px
   - Transition to 2-column or 1-column
   - Optimized touch targets
   - Adjusted typography scale

5. **Mobile**: 480px to 767px
   - Single column layouts
   - Full-width elements
   - Optimized spacing

6. **Mobile Small**: Below 480px
   - Full-width all elements
   - Minimal margins
   - Stacked layout
   - Prioritized content

## Summary

This design system creates a premium, elegant beauty and spa website that combines sophisticated typography (Raleway/Lato), a refined color palette (grays with purple accents), generous spacing, high-quality imagery, and fully responsive layouts. The design communicates luxury, tranquility, and professionalism—perfect for attracting discerning clients seeking premium wellness experiences.

The visual language is modern yet timeless, clean yet warm, professional yet inviting. Every element—from typography choices to color applications to spacing decisions—works together to create a cohesive, elegant experience that reflects the quality and sophistication of a high-end spa business.
